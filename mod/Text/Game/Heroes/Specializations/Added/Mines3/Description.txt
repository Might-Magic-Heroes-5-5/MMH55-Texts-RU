Эксперт по взрывчатым веществам: в начале боя герой создаёт «Огненные ловушки» вокруг отрядов Защитников гор, Воителей и Горных стражей (игнорируя группы по одному юниту). Значение героя на АТВ шкале сбрасывается до случайного значения между 0.8 и 0.9 после создания минного поля, умение «Острый ум» не действует на этого героя.
