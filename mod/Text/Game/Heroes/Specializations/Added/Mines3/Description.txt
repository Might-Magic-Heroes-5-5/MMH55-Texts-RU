Эксперт по взрывчатым веществам: в начале боя герой создаёт «Огненные ловушки» вокруг отрядов Демонов, Огненных демонов и Старших демонов (игнорируя группы по одному юниту). Значение героя на АТВ шкале сбрасывается до случайного значения между 0.6 и 0.7 после создания минного поля, умение «Острый ум» не действует на этого героя.
