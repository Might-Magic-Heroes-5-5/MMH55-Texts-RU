Герой изучает заклинание «Ледяная глыба». У заклинаний «Ледяная глыба», «Кольцо холода» и «Останавливающий холод» появляется замораживающий эффект.<br><color_default>Заклинания будут замораживать свои цели на 0.4 хода, в течение которых их значение на шкале инициативы не будет расти (эффективное снижение 0.4*«Инициатива»/10). Для «Кольца холода» эффект будет разделён между целями, получившими урон: 0.4/(Количество целей).
