Герой создает артефакт «Жезл» со случайным заклинанием для личного использования.<br>Герой учится растворять в бою мини-артефакты, переданные дружественным существам, для исцеления и воскрешения этих существ.<br><color_default>Восстанавливается 20*(Уровень Героя)*(1+Эф) ед. «Здоровья», где Эф - это количество эффектов, вложенных в растворяемый артефакт. Герой тратит 33% хода на растворение артефакта. Для каждого героя может быть создан только один «Жезл».
