Усовершенствованный ИИ не может использоваться для игры по сети. Для игры с ИИ по сети используйте MMH5.5_Utility.exe. Ванильный ИИ также может извлечь выгоду из выбора режима на экране выбора фракции. Чтобы принудительно установить режим, выберите один и тот же режим для всех игроков-людей. Получение опыта установлено так, чтобы сделать одинаковой скорость получения уровней, вне зависимости от сложности.
